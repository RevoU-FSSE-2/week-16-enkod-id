
## Installation

Install my-project with npm

```bash
  git clone 
  cd directoy 
  npm install
```
    
## API Reference
#### Register

```http
  POST /auth/register
```

#### Login

```http
  POST /auth/register
```

#### Get all Blog

```http
  GET /blog
```

#### Get item

```http
  GET /blog/${id}
```


#### Create 
```http
  POST /blog/
```
